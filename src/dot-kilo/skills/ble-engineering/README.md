# BLE Engineering

Bluetooth/BLE protocol analysis, reverse engineering, and implementation for ESP32, Android, and BlueZ platforms.

## Usage

Activates during BLE reverse engineering, GATT recovery, ESP32/Android/BlueZ development.

## Dependencies

- `bluetoothctl` — Bluetooth management
- `btmon` — BLE traffic monitoring
- `gatttool` — GATT operations
- `hcitool` — HCI commands
- `esptool` — ESP32 flashing

## Tools

- `.kilo/tools/ble-scan.ts` — BLE traffic capture
- `.kilo/tools/gatt-to-yaml.ts` — GATT JSON to YAML conversion
- `.kilo/tools/hex-analyzer.ts` — HEX dump analysis
