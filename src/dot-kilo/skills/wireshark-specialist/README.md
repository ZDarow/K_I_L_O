# Wireshark Specialist

Network traffic analysis: Wireshark/tshark, BLE, Bluetooth Classic, GATT, L2CAP, capture and filtering.

## Usage

Activates during Bluetooth/BLE traffic capture and analysis, protocol decoding, pcap parsing.

## Dependencies

- `wireshark` / `tshark` — traffic analysis
- `btmon` — BLE HCI capture
- `bluetoothctl` — Bluetooth management
- `tcpdump` — network traffic capture

## Resources

- `scripts/` — BLE capture, ATT extraction, Bluetooth analysis
- `references/` — BLE protocol, BT architecture, Wireshark filters
- `templates/` — report templates
