# APK Reverse Engineer

Декомпиляция APK и анализ Android-приложений с фокусом на Bluetooth (BLE/Classic).

## Использование

Активируется при задачах по реверс-инжинирингу Android-приложений, поиску Bluetooth-соединений, анализу трафика.

## Зависимости

- `apktool` — декомпиляция ресурсов
- `jadx` — декомпиляция в Java
- `grep`/`strings` — поиск сигнатур
- `keytool` — анализ сертификатов

## Ресурсы

- `scripts/` — анализ манифеста, декомпиляция, поиск Bluetooth, Frida-хуки
- `templates/` — чеклисты и шаблоны отчётов
