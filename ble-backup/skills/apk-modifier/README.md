# APK Modifier

Модификация APK-файлов: декомпиляция, анализ, внедрение кода, пересборка и подпись.

## Использование

Активируется автоматически при задачах по модификации APK.

## Зависимости

- [apktool](https://apktool.org/) — декомпиляция/пересборка
- [jadx](https://github.com/skylot/jadx) — декомпиляция в Java
- `keytool` — работа с ключами подписи
- `zipalign` — выравнивание APK
- `uber-apk-signer` — подпись APK

## Ресурсы

- `scripts/` — автоматизация типовых операций
- `templates/` — шаблоны отчётов
