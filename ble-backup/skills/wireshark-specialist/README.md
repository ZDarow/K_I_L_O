# Wireshark Specialist

Анализ сетевого трафика: Wireshark/tshark, BLE, Bluetooth Classic, GATT, L2CAP, захват и фильтрация.

## Использование

Активируется при задачах по захвату и анализу Bluetooth/BLE трафика, декодированию протоколов, разбору pcap.

## Зависимости

- `wireshark` / `tshark` — анализ трафика
- `btmon` — захват BLE HCI
- `bluetoothctl` — управление Bluetooth
- `tcpdump` — захват сетевого трафика

## Ресурсы

- `scripts/` — захват BLE, извлечение ATT, анализ Bluetooth
- `references/` — BLE-протокол, BT-архитектура, фильтры Wireshark
- `templates/` — шаблоны отчётов
