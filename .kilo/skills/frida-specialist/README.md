# Frida Specialist

Динамическая инструментация кода на Android/iOS/Linux/Windows — перехват вызовов, трассировка, обход защиты.

## Использование

Активируется при задачах по динамическому анализу, перехвату API-вызовов, обходу защиты, трассировке.

## Зависимости

- [frida](https://frida.re/) — фреймворк инструментации
- `frida-tools` — CLI-утилиты
- [objection](https://github.com/sensepost/objection) — mobile exploration

## Ресурсы

- `scripts/` — Frida-хуки для BLE, SSL-байпаса, перечисления
- `references/` — API-шпаргалки и паттерны хуков
- `templates/` — шаблоны отчётов
