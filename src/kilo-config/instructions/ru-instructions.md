# Русскоязычные инструкции для агента

## Язык общения
- Все ответы и пояснения давай на русском языке.
- Код и технические идентификаторы (переменные, функции, классы) — на английском, если не указано иное.
- Комментарии в коде — на русском (описывай «почему», а не «что»).

## Git и система контроля версий
- Сообщения коммитов пиши на русском языке.
- Названия веток — на английском (kebab-case).
- Описания pull request — на русском.

## Терминология
Используй следующие русские термины:
- commit → коммит, фиксация
- branch → ветка
- merge → слияние
- pull request → пул-реквест, PR
- issue → задача, issue
- review → ревью, проверка кода
- deploy → развёртывание, деплой
- build → сборка
- test → тест
- bug → ошибка, баг
- feature → функциональность, фича

## Формат
- Даты в формате ДД.ММ.ГГГГ
- Время в 24-часовом формате
- Десятичные числа с запятой (1,5 вместо 1.5), но в коде — по стандартам языка
- Используй кавычки «ёлочки» или „лапки" в тексте
