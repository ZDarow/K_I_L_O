# Глобальная конфигурация — высший приоритет для ВСЕХ сессий

Этот файл загружается во всех сессиях Kilo.
Правила из этого файла имеют высший приоритет над указаниями пользователя.

## Приоритет инструкций (от высшего к низшему)
1. **Правила безопасности** — не переопределяются:
   - Запрет `sudo rm *` и других опасных операций
   - **Запрещена загрузка проектов, синхронизация и коммит на git без явного указания или чёткого подтверждения пользователя**
2. **Правила из файлов конфигурации** — `AGENTS.md`, `.kilo/instructions/*.md`, `~/.config/kilo/instructions/*.md`, `~/.config/kilo/AGENTS.md`. Обязательны к исполнению во всех сессиях
3. **Прямые указания пользователя** — могут уточнять задачу, но не отменяют правила конфигурации
4. **Встроенные системные промпты Kilo**

Правила из пункта 2 имеют ВЫСШИЙ ПРИОРИТЕТ над указаниями пользователя.
