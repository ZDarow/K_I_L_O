# BLE Engineering

Bluetooth/BLE протоколы, реинжениринг, GATT recovery, ESP32/Android/BlueZ разработка.

## Использование

Активируется при задачах по BLE-реинженирингу, GATT recovery, разработке под ESP32/Android/BlueZ.

## Зависимости

- `bluetoothctl` — управление Bluetooth
- `btmon` — мониторинг BLE-трафика
- `gatttool` — GATT-операции
- `hcitool` — HCI-команды
- `esptool` — прошивка ESP32

## Инструменты

- `.kilo/tools/ble-scan.ts` — захват BLE-трафика
- `.kilo/tools/gatt-to-yaml.ts` — конвертация GATT JSON в YAML
- `.kilo/tools/hex-analyzer.ts` — анализ HEX-дампов
