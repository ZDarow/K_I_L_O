# BLE Engineering Skill

Используй этот skill когда задача связана с Bluetooth/BLE реинженирингом, анализом протоколов, GATT recovery, или реализацией BLE-драйверов под ESP32, Android, Linux (BlueZ).

## Процесс

1. Определи целевую платформу (ESP32/Android/BlueZ/все)
2. Настрой окружение под платформу
3. Выполни захват трафика или анализ существующих данных
4. Восстанови протокол/GATT-структуру
5. Реализуй совместимый код
6. Верифицируй
