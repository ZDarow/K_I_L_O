# APK Reverse Engineer

APK decompilation and Android application analysis with a focus on Bluetooth (BLE/Classic).

## Usage

Activates during Android reverse engineering, BLE connection search, traffic analysis.

## Dependencies

- `apktool` — resource decompilation
- `jadx` — Java decompilation
- `grep`/`strings` — signature search
- `keytool` — certificate analysis

## Resources

- `scripts/` — manifest analysis, decompilation, Bluetooth search, Frida hooks
- `templates/` — checklists and report templates
